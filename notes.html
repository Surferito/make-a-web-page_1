The Most Important Things to Remember

How the Web Works

The web is a collection of HTML documents. HTML stands for Hypertext Markup Language. The links between webs are called hyper links.

Elements and Tags

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. You dont have to remember all tags or attributes, just understand how HTML works and find the details when you need them. Nice list of Tags.

Inline and Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them (<DIV> is the typical example).

HTML - "Tree-Like Structure"

DOM stands for Document Object Model (tree-like structure). The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them (boxes inside of boxes inside of boxes...).

Avoiding Repetition

When programmers don't avoid repetition, they will often have to do the same thing over and over. This can lead to inefficiency and errors (when the programmer decides to make a change to something, he/she will have to remember to change it everywhere).

Example: If you give the same style to similar elements, you can reap the benefits of this when you quickly and easily modify all of them at once just by changing a single line of CSS.

CSS

CSS = Cascading Style Sheets. With CSS we can style the HTML elements.
Example: Give to similar HTML elements the same class name and then specify the style that should apply to that class.
